# Images de fond par section

Chaque section du portfolio a son propre fond d'image pour créer un effet "chaque scroll = slide d'un projet".

## Images requises

Placez les images suivantes dans ce dossier :

- `hero-bg.jpg` - Fond pour la section Hero
- `caps-bg.jpg` - Fond pour la section Multi-casquettes
- `work-bg.jpg` - Fond pour la section Projets
- `process-bg.jpg` - Fond pour la section Process
- `about-bg.jpg` - Fond pour la section À propos
- `contact-bg.jpg` - Fond pour la section Contact

## Recommandations

- **Format** : JPG ou WebP
- **Résolution** : 1920x1080 minimum (ou plus pour les écrans haute résolution)
- **Poids** : Optimiser les images (compression 80-85%)
- **Style** : Images abstraites, dégradés, textures qui s'harmonisent avec le design "liquid pop 2026"

## Fallback

Si une image n'est pas trouvée, le fond par défaut (grille + gradients) sera utilisé.
